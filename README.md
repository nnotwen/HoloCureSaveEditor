# HoloCureSaveEditor

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Made with Love](https://img.shields.io/badge/Made%20with-Love-red.svg)](https://github.com)

A browser-based save file editor for HoloCure. Edit your game saves directly in your web browser with an intuitive interface.

## Features

- **No Installation Required** - Access the editor directly from your web browser
- **Save File Management** - Load, edit, and download your HoloCure save files
- **User-Friendly Interface** - Intuitive UI for managing game data
- **Real-Time Editing** - Make changes and see results immediately

## Usage

1. Open the editor in your web browser
2. Import your HoloCure save file
3. Edit your save data using the interactive interface
4. Download and save your modified file back to your game directory

## How It Works

The editor runs entirely in your browser, meaning no data is sent to external servers. Your save files remain private and secure.
