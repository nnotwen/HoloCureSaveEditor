Privacy Policy

Data Collection

This tool does not collect, store, or transmit any personal data. All save file processing happens entirely in your browser, meaning no files are uploaded to any server.

Local Storage

Snapshots you create are stored locally in your browser's IndexedDB. This data never leaves your device and can be cleared at any time through your browser settings.

Comments (Giscus)

This site uses Giscus, a commenting system powered by GitHub Discussions. To leave a comment, you need a GitHub account. When you comment:

  • Your GitHub username, avatar, and comment text are stored in this repository's Discussions section and are publicly visible.
  • Giscus stores a server-encrypted token in your browser's localStorage to keep you signed in. No cookies are used for this feature.
  • You can delete this token at any time by clearing your browser's local storage.

For more information on how GitHub handles your data, please review GitHub's Privacy Statement .

Third-Party Services

Aside from Giscus (which loads comment data from GitHub), this tool uses no analytics, tracking cookies, or external services. All libraries are bundled locally and run entirely in your browser. No save file data is ever sent to any external servers.

Your Rights

If you have left comments, you can request deletion by contacting us. Note that comments are stored on GitHub, so you may also need to contact GitHub to exercise your rights regarding your account data. For save files and snapshots, all data remains under your control on your own device.
